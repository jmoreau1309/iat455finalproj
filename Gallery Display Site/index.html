  • About This Project

    Nathan Cerone & Jennifer Moreau created this site and associated functionalities as their final project for IAT 455 Spring 2023. This project applies the image quilting process to recreate real life photos into certain art styles. By inputting an image and selecting an art style, that image can appear as if it was painted or created by that specific artist using their recognizable style. This not only gives users the capability to use this computational method, but will also preserve art styles and inspire more appreciation for them.

Designed & Developed by Nathan Cerone & Jennifer Moreau
